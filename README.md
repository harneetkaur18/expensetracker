Project title - Expense Trcker

What is this project?
This is a Python-based Expense Tracker mini project designed to help users record and track their daily expenses in a simple and efficient way using a command-line interface.

What does it do?
The Expense Tracker allows users to:
1. Add new expenses
2. View all recorded expenses
3. View the total amount spent
The project focuses on applying Python fundamentals to solve a real-life problem.

How can someone run it?
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Run the Python file
4. Follow the on-screen instructions to add and view expenses
